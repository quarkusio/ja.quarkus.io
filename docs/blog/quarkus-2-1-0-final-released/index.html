July 29, 2021 #release

Quarkus 2.1.0.Finalリリース - Dev Service for Keycloak、Reactive SQL Serverエクステンション、Kotlin 1.5

By Guillaume Smet

本日、Quarkus 2.1.0.Finalをリリースしました。多くの改良と改善、いくつかの新機能が含まれています。

  • Dev Service for Keycloak

  • リアクティブなMicrosoft SQL Serverエクステンション

  • Kotlin 1.5へのアップグレード

マイグレーションガイド

2.0からの移行は、 移行ガイドをご参照ください。

変更点

Dev Service for Keycloak

Dev Servicesは、アプリケーションの開発やテストを行う際に、必要なコンテナを自動的に起動します。

多くのQuarkusのエクステンションがすでにカバーされており、2.1では Dev Service for Keycloakを追加しました。

リアクティブなMicrosoft SQL Serverエクステンション

2.1では、Microsoft SQL Serverのリアクティブ・ドライバー・エクステンションが追加されています - quarkus-reactive-mssql-client 。

Kotlin 1.5

Kotlinエクステンションとツールを1.5にアップグレードしました（1.5.21）。

完全な変更履歴

2.1.0.CR1および 2.1.0.Finalの完全な変更履歴はGitHubでご覧いただけます。
