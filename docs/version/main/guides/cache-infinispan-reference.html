Infinispan гӮӯгғЈгғғгӮ·гғҘ

гғҮгғ•гӮ©гғ«гғҲгҒ§гҒҜгҖҒQuarkus Cache гҒҜгғҗгғғгӮҜгӮЁгғігғүгҒЁгҒ—гҒҰ Caffeine гӮ’дҪҝз”ЁгҒ—гҒҫгҒҷгҖӮ д»ЈгӮҸгӮҠгҒ« Infinispan гӮ’дҪҝз”ЁгҒҷгӮӢгҒ“гҒЁгӮӮеҸҜиғҪгҒ§гҒҷгҖӮ

гҒ“гҒ®жҠҖиЎ“гҒҜгҖҒpreviewгҒЁиҖғгҒҲгӮүгӮҢгҒҰгҒ„гҒҫгҒҷгҖӮ

preview гҒ§гҒҜгҖҒдёӢдҪҚдә’жҸӣжҖ§гӮ„гӮЁгӮігӮ·гӮ№гғҶгғ гҒ§гҒ®еӯҳеңЁгҒҜдҝқиЁјгҒ•гӮҢгҒҰгҒ„гҒҫгҒӣгӮ“гҖӮе…·дҪ“зҡ„гҒӘж”№е–„гҒ«гҒҜиЁӯе®ҡгӮ„ API гҒ®еӨүжӣҙгҒҢеҝ…иҰҒгҒ«гҒӘгӮӢгҒӢгӮӮгҒ—гӮҢгҒҫгҒӣгӮ“гҒҢгҖҒ stable гҒ«гҒӘгӮӢгҒҹгӮҒгҒ®иЁҲз”»гҒҜзҸҫеңЁйҖІиЎҢдёӯгҒ§гҒҷгҖӮгғ•гӮЈгғјгғүгғҗгғғгӮҜгҒҜ гғЎгғјгғӘгғігӮ°гғӘгӮ№гғҲ гӮ„ GitHub гҒ®иӘІйЎҢз®ЎзҗҶ гҒ§еҸ—гҒ‘д»ҳгҒ‘гҒҰгҒ„гҒҫгҒҷгҖӮ

гҒЁгӮҠгҒҶгӮӢгӮ№гғҶгғјгӮҝгӮ№гҒ®е®Ңе…ЁгҒӘгғӘгӮ№гғҲгҒ«гҒӨгҒ„гҒҰгҒҜгҖҒ FAQгҒ®й …зӣ® гӮ’еҸӮз…§гҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ

гӮӯгғЈгғғгӮ·гғҘгғҗгғғгӮҜгӮЁгғігғүгҒЁгҒ—гҒҰгҒ® Infinispan

Quarkus гӮӯгғЈгғғгӮ·гғҘгҒ®гғҗгғғгӮҜгӮЁгғігғүгҒЁгҒ—гҒҰ Infinispan гӮ’дҪҝз”ЁгҒҷгӮӢе ҙеҗҲгҖҒгӮӯгғЈгғғгӮ·гғҘгҒ•гӮҢгҒҹеҗ„гӮўгӮӨгғҶгғ гҒҜ Infinispan гҒ«дҝқеӯҳгҒ•гӮҢгҒҫгҒҷгҖӮ

  • гғҗгғғгӮҜгӮЁгғігғүгҒҜ (еҲҘгҒ®иЁӯе®ҡгҒҢгҒ•гӮҢгҒҰгҒ„гҒӘгҒ„йҷҗгӮҠ) <default> Infinispan гӮҜгғ©гӮӨгӮўгғігғҲгӮ’дҪҝз”ЁгҒҷгӮӢгҒҹгӮҒгҖҒйҒ©еҲҮгҒ« гӮ»гғғгғҲгӮўгғғгғ—гҒ•гӮҢгҒҰгҒ„гӮӢгҒ“гҒЁгӮ’зўәиӘҚгҒ—гҒҰгҒҸгҒ гҒ•гҒ„ (гҒҫгҒҹгҒҜгҖҒInfinispan Dev Service гӮ’дҪҝз”ЁгҒ—гҒҰгҒҸгҒ гҒ•гҒ„)гҖӮ

Infinispan гғҗгғғгӮҜгӮЁгғігғүгҒ®иЁӯе®ҡ

Infinispan гғҗгғғгӮҜгӮЁгғігғүгҒҜ <default> Infinispan гӮҜгғ©гӮӨгӮўгғігғҲгӮ’дҪҝз”ЁгҒ—гҒҫгҒҷгҖӮ Infinispan гҒёгҒ®гӮўгӮҜгӮ»гӮ№гӮ’иЁӯе®ҡгҒҷгӮӢгҒ«гҒҜгҖҒInfinispan гғӘгғ•гӮЎгғ¬гғігӮ№ гӮ’еҸӮз…§гҒ—гҒҰгҒҸгҒ гҒ•гҒ„гҖӮ

й–ӢзҷәгғўгғјгғүгҒ§гҒҜгҖҒInfinispan Dev Service гӮ’дҪҝз”ЁгҒ§гҒҚгҒҫгҒҷгҖӮ

гӮӯгғЈгғғгӮ·гғҘгҒ«еҲҘгҒ® Infinispan гӮ’дҪҝз”ЁгҒҷгӮӢе ҙеҗҲгҒҜгҖҒ client-name гӮ’д»ҘдёӢгҒ®гӮҲгҒҶгҒ«иЁӯе®ҡгҒ—гҒҫгҒҷгҖӮ

quarkus.cache.infinispan.client-name=another

гғһгғјгӮ·гғЈгғӘгғігӮ°

Quarkus гҒ§ Infinispan гҒЁеҜҫи©ұгҒҷгӮӢе ҙеҗҲгҖҒгӮӯгғЈгғғгӮ·гғҘгҒ«гғҮгғјгӮҝгӮ’дҝқеӯҳгҒ—гҒҹгӮҠгҖҒгӮӯгғЈгғғгӮ·гғҘгҒӢгӮүгғҮгғјгӮҝгӮ’еҸ–еҫ—гҒҷгӮӢйҡӣгҒ«гҖҒJava гғ—гғӘгғҹгғҶгӮЈгғ–еһӢгӮ’з°ЎеҚҳгҒ«гӮ·гғӘгӮўгғ«еҢ–гҒҠгӮҲгҒігғҮгӮ·гғӘгӮўгғ«еҢ–гҒ§гҒҚгҒҫгҒҷгҖӮInfinispan гҒ«гҒҜиҝҪеҠ гҒ®гғһгғјгӮ·гғЈгғӘгғігӮ°иЁӯе®ҡгҒҜеҝ…иҰҒгҒӮгӮҠгҒҫгҒӣгӮ“гҖӮ

@CacheResult(cacheName = "weather-cache") (1)
public String getDailyForecast(String dayOfWeek, int dayOfMonth, String city) { (2)
    return dayOfWeek + " will be " + getDailyResult(dayOfMonth % 4) + " in " + city;
}
1 гҒ“гҒ®гғЎгӮҪгғғгғүгҒ®е®ҹиЎҢгӮ’ 'weather-cache' гҒ«гӮӯгғЈгғғгӮ·гғҘгҒҷгӮӢгӮҲгҒҶжҢҮе®ҡгҒ—гҒҫгҒҷгҖӮ
2 гӮӯгғјгҒҜгҖҒ String dayOfWeekгҖҒ int dayOfMonthгҖҒгҒҠгӮҲгҒі String city гӮ’зө„гҒҝеҗҲгӮҸгҒӣгҒҹгӮӮгҒ®гҒ§гҒҷгҖӮй–ўйҖЈд»ҳгҒ‘гӮүгӮҢгҒҰгҒ„гӮӢеҖӨгҒҜ String еһӢгҒ§гҒҷгҖӮ

Quarkus гҒҜгғҮгғ•гӮ©гғ«гғҲгҒ§ Infinispan гҒ«гҒҠгҒ‘гӮӢгғҮгғјгӮҝгҒ®гӮ·гғӘгӮўгғ«еҢ–гҒ« Protobuf гӮ’дҪҝз”ЁгҒ—гҒҫгҒҷгҖӮInfinispan гҒҜ гғҮгғјгӮҝгӮ’ж§ӢйҖ еҢ–гҒҷгӮӢж–№жі• гҒЁгҒ—гҒҰ Protobuf гӮ’жҺЁеҘЁгҒ—гҒҰгҒ„гҒҫгҒҷгҖӮгҒ—гҒҹгҒҢгҒЈгҒҰгҖҒPlain Old Java Objects (POJO) гӮ’дҪҝз”ЁгҒҷгӮӢе ҙеҗҲгҖҒгғҰгғјгӮ¶гғјгҒҜ Infinispan гҒ§гҒ® гғһгғјгӮ·гғЈгғӘгғігӮ°з”ЁгҒ®гӮ№гӮӯгғјгғһгӮ’жҸҗдҫӣгҒҷгӮӢеҝ…иҰҒгҒҢгҒӮгӮҠгҒҫгҒҷгҖӮ

Java еһӢгҒ®гғһгғјгӮ·гғЈгғӘгғігӮ°

java.time.LocalDate гӮ’дҪҝз”ЁгҒ—гҒҰиӨҮеҗҲгӮӯгғјгӮ’дҪңжҲҗгҒҷгӮӢгҒЁгҒ—гҒҫгҒҷгҖӮ

@CacheResult(cacheName = "weather-cache") (1)
public String getDailyForecast(LocalDate date, String city) { (2)
    return date.getDayOfWeek() + " will be " + getDailyResult(date.getDayOfMonth() % 4) + " in " + city;
}
1 гҒ“гҒ®гғЎгӮҪгғғгғүе®ҹиЎҢгҒ®еҝңзӯ”гӮ’ 'weather-cache' гҒ«гӮӯгғЈгғғгӮ·гғҘгҒҷгӮӢгӮҲгҒҶгҒ«гғӘгӮҜгӮЁгӮ№гғҲгҒ—гҒҫгҒҷгҖӮ
2 гӮӯгғјгҒҜгҖҒ java.util.LocalDate гҒ®ж—Ҙд»ҳгҒЁ String city гӮ’зө„гҒҝеҗҲгӮҸгҒӣгҒҹгӮӮгҒ®гҒ§гҒҷгҖӮй–ўйҖЈд»ҳгҒ‘гӮүгӮҢгҒҹеҖӨгҒҜ 'String' еһӢгҒ§гҒҷгҖӮ

Infinispan гҒҜгғҮгғ•гӮ©гғ«гғҲгҒ§ Quarkus гҒ® Protobuf гӮ’дҪҝз”ЁгҒ—гҒҰгғҮгғјгӮҝгӮ’гӮ·гғӘгӮўгғ«еҢ–гҒҷгӮӢгҒҹгӮҒгҖҒгӮігғјгғүгӮ’е®ҹиЎҢгҒҷгӮӢгҒЁд»ҘдёӢгҒ®гӮЁгғ©гғјгҒҢзҷәз”ҹгҒ—гҒҫгҒҷгҖӮ

java.lang.IllegalArgumentException:
No marshaller registered for object of Java type java.time.LocalDate

Protobuf гҒҜгҖҒ java.time.LocalDate гӮ’гғһгғјгӮ·гғЈгғӘгғігӮ°гҒҷгӮӢж–№жі•гӮ’жң¬жқҘгҒҜиӘҚиӯҳгҒ—гҒҰгҒ„гҒҫгҒӣгӮ“гҖӮе№ёгҒ„гҒӘгҒ“гҒЁгҒ«гҖҒProtostream гҒҜ @ProtoAdapter гҒЁ @ProtoSchema гӮ’дҪҝз”ЁгҒ—гҒҰгҖҒгҒ“гҒ®е•ҸйЎҢгҒ«еҜҫгҒҷгӮӢз°ЎеҚҳгҒӘи§Јжұәзӯ–гӮ’жҸҗдҫӣгҒ—гҒҫгҒҷгҖӮ

@ProtoAdapter(LocalDate.class)
public class LocalDateAdapter {
    @ProtoFactory
    LocalDate create(String localDate) {
        return LocalDate.parse(localDate);
    }

    @ProtoField(1)
    String getLocalDate(LocalDate localDate) {
        return localDate.toString();
    }
}

@ProtoSchema(includeClasses = LocalDateAdapter.class, schemaPackageName = "quarkus")
public interface Schema extends GeneratedSchema {
}

Infinispan гҒ§гҒҜгҖҒlifespan гҒҜиЁӯе®ҡгғ‘гғ©гғЎгғјгӮҝгғјгҒ® 1 гҒӨгҒ§гҒӮгӮҠгҖҒгӮЁгғігғҲгғӘгғј (гҒҫгҒҹгҒҜгӮӘгғ–гӮёгӮ§гӮҜгғҲ) гҒҢгҖҒ дҪңжҲҗгҒҫгҒҹгҒҜжңҖзөӮгӮўгӮҜгӮ»гӮ№гҒ•гӮҢгҒҰгҒӢгӮүжңҹйҷҗеҲҮгӮҢгҒЁиҰӢгҒӘгҒ•гӮҢгӮӯгғЈгғғгӮ·гғҘгҒӢгӮүеүҠйҷӨгҒ•гӮҢгӮӢгҒҫгҒ§гҖҒ гӮӯгғЈгғғгӮ·гғҘгҒ«ж®ӢгӮӢгҒ“гҒЁгҒҢгҒ§гҒҚгӮӢжңҖеӨ§жҷӮй–“гӮ’жұәе®ҡгҒ—гҒҫгҒҷгҖӮ

Infinispan гӮӯгғЈгғғгӮ·гғҘгҒ®гӮЁгғігғҲгғӘгғјгҒ« lifespan гғ‘гғ©гғЎгғјгӮҝгғјгӮ’иЁӯе®ҡгҒҷгӮӢгҒЁгҖҒ жңҹй–“гӮ’жҢҮе®ҡгҒ—гҒҫгҒҷгҖӮгӮЁгғігғҲгғӘгғјгҒҢгӮӯгғЈгғғгӮ·гғҘгҒ«иҝҪеҠ гҒ•гӮҢгӮӢгҒӢгӮўгӮҜгӮ»гӮ№гҒ•гӮҢгҒҹ (иӘӯгҒҝеҸ–гӮҠгҒҫгҒҹгҒҜжӣёгҒҚиҫјгҒҝ) еҫҢгҖҒгҒқгҒ®гӮЁгғігғҲгғӘгғјгҒ®еӯҳз¶ҡжңҹй–“гҒ®гӮ«гӮҰгғігғҲгғҖгӮҰгғігҒҢе§ӢгҒҫгӮҠгҒҫгҒҷгҖӮгӮЁгғігғҲгғӘгғјгҒ® дҪңжҲҗгҒҫгҒҹгҒҜжңҖзөӮгӮўгӮҜгӮ»гӮ№гҒӢгӮүгҒ®зөҢйҒҺжҷӮй–“гҒҢгҖҒжҢҮе®ҡгҒ•гӮҢгҒҹгҖҢеӯҳз¶ҡжңҹй–“гҖҚгӮ’и¶…гҒҲгҒҹе ҙеҗҲгҖҒгҒқгҒ®гӮЁгғігғҲгғӘгғјгҒҜ жңҹйҷҗеҲҮгӮҢгҒЁиҰӢгҒӘгҒ•гӮҢгҖҒгӮӯгғЈгғғгӮ·гғҘгҒӢгӮүеүҠйҷӨгҒ®еҜҫиұЎгҒЁгҒӘгӮҠгҒҫгҒҷгҖӮ

quarkus.cache.infinispan.my-cache.lifespan=10s

Max Idle

Infinispan гӮӯгғЈгғғгӮ·гғҘгҒ®гӮЁгғігғҲгғӘгғјгҒ« max-idle гғ‘гғ©гғЎгғјгӮҝгғјгӮ’иЁӯе®ҡгҒҷгӮӢе ҙеҗҲгҖҒжҷӮй–“гӮ’жҢҮе®ҡгҒ—гҒҫгҒҷгҖӮ гӮӯгғЈгғғгӮ·гғҘеҶ…гҒ®гӮЁгғігғҲгғӘгғјгҒҢгӮўгӮҜгӮ»гӮ№ (иӘӯгҒҝеҸ–гӮҠгҒҫгҒҹгҒҜжӣёгҒҚиҫјгҒҝ) гҒ•гӮҢгҒҹеҫҢгҖҒжҢҮе®ҡгҒ•гӮҢгҒҹжңҹй–“еҶ…гҒ« гҒқгҒ®гӮЁгғігғҲгғӘгғјгҒ«еҜҫгҒ—гҒҰгҒқгҒ®еҫҢгӮўгӮҜгӮ»гӮ№гҒҢгҒӘгҒ„е ҙеҗҲгҒҜгҖҒгӮўгӮӨгғүгғ«зҠ¶ж…ӢгҒЁгҒҝгҒӘгҒ•гӮҢгҒҫгҒҷгҖӮгӮўгӮӨгғүгғ«жҷӮй–“гҒҢ max-idle жңҹй–“гӮ’и¶…гҒҲгӮӢгҒЁгҖҒгӮЁгғігғҲгғӘгғјгҒҜжңҹйҷҗеҲҮгӮҢгҒЁгҒҝгҒӘгҒ•гӮҢгҖҒгӮӯгғЈгғғгӮ·гғҘгҒӢгӮүгҒ®гӮЁгғ“гӮҜгӮ·гғ§гғігҒ® еҜҫиұЎгҒЁгҒӘгӮҠгҒҫгҒҷгҖӮ
